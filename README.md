# AI Career Guidance Platform - Frontend

A modern React-based frontend for an AI-powered Career Guidance Platform.

## Features

* User Authentication
* Resume Upload Interface
* Career Dashboard
* Responsive UI
* Sidebar Navigation
* AI Career Recommendations
* Modern React + Vite Architecture

## Tech Stack

* React
* Vite
* JavaScript
* CSS
* React Router DOM

## Installation

Clone the repository:

```bash
git clone https://github.com/sharwari0109/ai-career-guidance-frontend.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
